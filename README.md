# Brustkrebs-Klassifizierung 🔬

Dieses Projekt ist im Rahmen meines Informatik-Studiums an der 
Hochschule Hamm-Lippstadt entstanden. Die Idee war, den 
Brustkrebs-Klassifizierungsalgorithmus, den ich für das Modul 
Machine Learning selbst implementiert habe, durch eine 
Web-Anwendung für andere zugänglich zu machen.

## Was die App kann
- Brustkrebs-Befunde als gutartig oder bösartig klassifizieren
- Patienten anlegen und Ergebnisse speichern
- Datenanalyse mit Visualisierungen anzeigen
- Nutzerverwaltung mit Login und Registrierung

## Starten

```bash
pip install -r requirements.txt
python app.py
```

Dann im Browser: http://127.0.0.1:5000
